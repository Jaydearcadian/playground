# Dune Verification Layer

Use this skill for all Dune witness artifacts.

## Responsibilities
- Generate/maintain parameterized Dune SQL.
- Maintain dashboard spec mapping docs.
- Map Dune outputs to manifest/evidence summary fields.
- Implement/adjust Dune query execution clients.

## Workflow
1. Define query purpose and required parameters.
2. Ensure output columns map to observatory fields.
3. Update dashboard spec.
4. Add example usage for API client execution.
